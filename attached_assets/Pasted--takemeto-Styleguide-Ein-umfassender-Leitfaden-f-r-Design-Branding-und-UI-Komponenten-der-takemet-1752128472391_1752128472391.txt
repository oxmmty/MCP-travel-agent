# takemeto Styleguide

Ein umfassender Leitfaden für Design, Branding und UI-Komponenten der takemeto Plattform.

## Markenidentität

### Logo
- **Primäres Logo**: Freigestelltes PNG mit Gradient-Schriftart
- **Verwendung**: Navigation (h-10), Footer (h-10)
- **Alt-Text**: "takemeto"
- **Datei**: `bbdefd7d-6de9-429e-a2eb-a0d86d64cbb0_1752127901407.png`

### Farbpalette

#### Primäre Farben
```css
--coral-primary: hsl(8, 85%, 62%)     /* #E85A4F - Coral */
--purple-mid: hsl(288, 65%, 55%)      /* #A855A3 - Purple */
--blue-end: hsl(213, 65%, 58%)        /* #3B82F6 - Blue */
```

#### Gradient-Definitionen
```css
/* Haupt-Gradient (Coral → Purple → Blue) */
--primary-gradient-start: hsl(8, 85%, 62%)
--primary-gradient-end: hsl(213, 65%, 58%)

/* Akzent-Gradient (Coral → Purple) */
--accent-gradient-start: hsl(8, 85%, 62%)
--accent-gradient-end: hsl(288, 65%, 55%)
```

#### Systemfarben
```css
--background: hsl(0, 0%, 100%)        /* Weiß */
--foreground: hsl(222, 84%, 4.9%)     /* Dunkelblau */
--muted: hsl(210, 40%, 98%)           /* Hellgrau */
--muted-foreground: hsl(215, 13.8%, 34.1%) /* Mittelgrau */
--border: hsl(214, 32%, 91.2%)        /* Rahmengrau */
--primary: hsl(8, 85%, 62%)           /* Coral (für UI-Komponenten) */
--ring: hsl(8, 85%, 62%)              /* Focus-Ring */
```

## Typografie

### Schriftarten

#### Haupt-Schriftart
```css
font-family: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif;
```
- **Verwendung**: Fließtext, UI-Komponenten, Navigation
- **Gewichte**: 400 (normal), 500 (medium), 600 (semibold), 700 (bold)

#### Gradient-Schriftart
```css
font-family: 'Noto Serif', Georgia, serif;
```
- **Verwendung**: Markennamen, Logo-Text, besondere Hervorhebungen
- **Gewichte**: 400 (normal), 500 (medium), 600 (semibold), 700 (bold)
- **Import**: `@import url('https://fonts.googleapis.com/css2?family=Noto+Serif:wght@400;500;600;700&display=swap');`

### Text-Hierarchie

#### Überschriften
```css
/* H1 - Hero */
font-size: 2xl sm:3xl md:4xl lg:5xl xl:6xl
font-weight: light (300) bis medium (500)
line-height: tight

/* H2 - Sektion */
font-size: 3xl md:4xl
font-weight: medium (500)

/* H3 - Untersektion */
font-size: 2xl
font-weight: semibold (600)

/* H4 - Komponenten */
font-size: xl
font-weight: medium (500)
```

#### Fließtext
```css
/* Haupt-Paragraph */
font-size: base md:lg lg:xl
font-weight: normal (400)
line-height: relaxed

/* Klein-Text */
font-size: sm
color: muted-foreground
```

## Verwendungsrichtlinien

### Logo-Verwendung
1. **Navigation**: Immer h-10, links positioniert
2. **Footer**: h-10, mit mb-4 Abstand
3. **Niemals**: Logo verzerren oder Proportionen ändern

### Gradient-Verwendung
1. **Markennamen**: Immer mit `.text-gradient` Klasse
2. **CTAs**: Primär-Buttons mit `bg-gradient-primary`
3. **Akzente**: Icons und Highlights mit Gradient-Hintergründen

### Typografie-Regeln
1. **Noto Serif**: Nur für Markennamen und besondere Hervorhebungen
2. **Inter**: Für alle anderen Texte
3. **Kontrast**: Immer ausreichend für Barrierefreiheit

### Responsive Design
- **Mobile First**: Alle Designs beginnen bei kleinsten Bildschirmen
- **Breakpoints**: sm (640px), md (768px), lg (1024px), xl (1280px)
- **Logo**: Bleibt immer h-10, passt sich automatisch an
